## Array Operation in Javascript

# Project Overview

This project demonstrate basic array operations in javascript using functions . The program performs the following tasks on a dummy array.

->Find the maximum number in the array.
->Calculate the sum of all elements.
->Counts the number of odd numbers.
->Displays the results in the browser console

## Technologies used

-HTML
-Javascript

## How to Run
1.Clone the folder
2.open the folder in vs code
3.Make sure index.html and style.css are in the same folder.
4.Open index.html in any browser.
